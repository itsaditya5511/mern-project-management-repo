🧠 About Project

«🚀 A futuristic collaborative project management platform built using the MERN Stack
Inspired by tools like Jira & Trello, enabling teams to manage workflows efficiently.»

---

⚙️ Core Features

🔐 Authentication & Authorization

- JWT Authentication 🔑
- Role-based system:
  - 🛡 Admin
  - 📊 Manager
  - 👨‍💻 Member
- Protected routes (Frontend + Backend)

---

🏢 Workspace System

- Create multiple workspaces
- Invite members via email
- Assign roles dynamically
- Manage team members
- Multi-project environment
